My Work

These are my 3 best pieces of work. They display my skills and challenges I had to face and overcome.


Command Structure



Command Structure is a Magic the Gathering commander deck building website I have created for my Javascript class. In this site you are able to create your own decks and search for cards to put in those decks. The deck can be renamed and deleted, and cards can be added and deleted from decks as well. You must however create a profile on the website in order to create decks from the Add New Deck form. Your profile information and decks is stored in a Firestore database

Boardwalk Boardgames



Boardwalk Boardgames was created in my PHP class to demonstrate my skills and abilities in using PHP and the back-end of web development. In this site you are able to add and delete boardgames from the bitlamps database keeping track of the games. You are also able to view a list of orders made by different customers.

Author Website



The Author website was a project from my Wordpress class. This project was my introduction to using Wordpress for making websites. Wordpress is a platform that is another method of making websites by utilizing themes and plugins rather than just coding in a text editor.
